# 📎 SOORE(팀명: SOORE) - BPS 업무 관리 프로그램
![main](https://github.com/2022-SMHRD-IS-BigData3/Soore/assets/98467249/3366859f-da30-4573-a1e7-9bda55bee41e)


## 👀 서비스 소개
* 서비스명:  BPS(업무관리프로그램)
* 서비스설명: 프리랜서들을 위한 업무관리 프로그램
<br>

## 📅 프로젝트 기간
2022.04.22 ~ 2022.06.03 (6주)
<br>

## ⭐ 주요 기능
* GANTT CHART
* CALENDAR
* KANBAN BOARD
* WORKSPACE
<br>
